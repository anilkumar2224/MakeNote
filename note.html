Title And more great features! - important notes, checklists, reminders. Add notes Tap the plus icon at the bottom to create a new note. You can edit the notes by taping the corresponding note grid item. Create checklists Checkbox is an essential element of a to-do list. Tap the checkbox icon at the bottom of a note to add a checkbox to a paragraph of text. Check the box when the stuff is done. Set reminders Set reminders directly from notes. Tap the "More" button in the upper right corner of a note, select "Reminders", set the time.
